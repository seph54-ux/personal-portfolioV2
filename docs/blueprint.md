# **App Name**: DesignFlow Portfolio

## Core Features:

- Project Showcase: Display portfolio pieces categorized by type (Posters, Logos, OBS Overlays, Web UI Design, Video Projects, Websites/Webapps).
- Logo Variant Display: Showcase different logo variants for each brand.
- Interactive OBS Overlay Display: Show both the overlay assets and examples of the overlay in action within OBS.
- Tech Stack Display: List the tools, frameworks, and technologies used in each project.
- Blog Integration: Display blog posts related to design projects or industry insights.
- Affiliate Showcase: Showcase affiliated marketplaces where design assets can be purchased, potentially using AI to suggest relevant products based on portfolio content.
- Contact Form: Include a contact form with a call-to-action button.

## Style Guidelines:

- Light Mode: Primary color is reddish (#E07A5F) for a warm and inviting feel. Background color is light (#F2E8DF).
- Dark Mode: Primary color is yellowish (#F2CC8F) for contrast against the dark background. Background color is very dark blue (#1E2022).
- Accent color: Teal (#3D405B) to complement both reddish and yellowish primary colors.
- Font pairing: 'Poppins' (sans-serif) for headlines and short amounts of body text, and 'PT Sans' (sans-serif) for longer body text to ensure readability.
- Use a grid-based layout for a clean and organized presentation of portfolio items.
- Subtle transitions and animations on hover to enhance user experience without being distracting.
- Use a consistent and modern icon set for navigation and to represent different project categories.
