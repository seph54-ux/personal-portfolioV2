# Philjoseph Orlina - Personal Portfolio

**Tagline:** Transforming Ideas into Digital Reality.

This repository contains the source code for my personal portfolio website, a dynamic and fully responsive platform designed to showcase my skills and projects in design and development. Built with a modern tech stack, this portfolio serves as a central hub for my work, thoughts, and professional services.

---

## ‚ú® Features

- **Fully Responsive Design:** Optimized for a seamless experience on desktops, tablets, and mobile devices.
- **Project Showcase:** Multiple, organized sections for different categories of work, including:
  - Web UI & App Design
  - Logo & Poster Design
  - OBS Overlays for Streaming
  - Full-Stack Websites & Webapps
- **Interactive UI:** Features like image/video modals, carousels, and custom animations (like the animated stats) to create an engaging user experience.
- **Dynamic Content Management:** Project and blog data are managed centrally, making the site easy to update.
- **Integrated Blog:** A complete blog with individual article pages to share insights and stories.
- **Secure Contact Form:** A functional contact form using EmailJS and protected by Google reCAPTCHA v3.
- **Dark/Light Mode:** Theme toggling for user preference, powered by `next-themes`.
